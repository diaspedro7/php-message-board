# Message Board

## Sobre o projeto

Este projeto consiste em um **mural de mensagens simples**, onde o usuário pode enviar um nome e uma mensagem através de um formulário, e as mensagens enviadas são exibidas em uma lista na mesma página.

Os dados são processados no back-end com **PHP** e armazenados em um arquivo **JSON**, permitindo persistência das mensagens sem uso de banco de dados.

---

## Objetivo do projeto

O principal objetivo deste projeto é praticar conceitos fundamentais do desenvolvimento web, incluindo:

- Integração entre front-end e back-end  
- Envio e recebimento de dados via **HTTP POST**  
- Manipulação de arquivos **JSON** com PHP  
- Organização e abstração de código utilizando funções  
- Estruturação de layout com **HTML e CSS**  
- Exibição dinâmica de dados no front-end  

Este projeto foi desenvolvido com foco em aprendizado e prática.

---

## Ferramentas utilizadas

- **VS Code** — desenvolvimento do código  
- **XAMPP** — servidor local (Apache + PHP)  
- **Navegador Brave** — testes e visualização do projeto  

---

## Como executar o projeto

1. Instale o **XAMPP**
2. Copie a pasta do projeto para o diretório `htdocs`
3. Inicie o servidor **Apache**
4. Acesse no navegador: http://localhost/nome-da-pasta-do-projeto
